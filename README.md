# ⚡ FISI Toolbox v1.0.0
**All-in-one IT toolbox for FISI professionals**  
by Cedric Thieme

---

## 📦 Installation

```bash
unzip fisi-toolbox.zip
cd fisi-toolbox
sudo bash install.sh
```

Das war's. Die App erscheint danach automatisch in der App-Suche.

## 🚀 Starten

- **App-Suche:** Nach `FISI Toolbox` suchen
- **Terminal:** `fisi-toolbox`

## 🗑️ Deinstallieren

```bash
sudo bash /opt/fisi-toolbox/uninstall.sh
```

## 🐧 Unterstützte Systeme

- Fedora 38+
- Debian 11+
- Ubuntu 22.04+
- Linux Mint, Pop!_OS, Zorin OS und weitere Debian/Ubuntu-Derivate
